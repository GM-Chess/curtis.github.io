Projects

Microcontrollers are the foundation of an embedded system. I created an array keyboard with an XMC4500 microcontroller development board. This MCU has an ARM Cortex-M4 the processor on it, which allowed me to excel in the ARM Cortex instruction set. I program the MCU with C in the DAVE V4, an Infineon IDE. The project improved my embedded C knowladge because I used SysTic Interrupts and pull-up/down switches. For my code, please visit my GitHub

I choose this topic as it is crucial for big corporations to demonstrate good ethical practices. Next, I created a machine learning(ML) algorithm to highlight the biases in ML.
